### Community плагины которые у меня задействованы:
**Calendar by Lian Caim** - удобен тем, что можно нажав на дату, открыть заметку по этой дате, если она создавалась

<img width="363" height="291" alt="image" src="https://github.com/user-attachments/assets/6896db0a-b4fb-4421-9d31-1323118b9e62" />

**Excalidraw** - удобен, если нужно набросать какие то мысли в рисунок, очень приятный ui/ux даже для мыши

<img width="1260" height="1107" alt="image" src="https://github.com/user-attachments/assets/8e1923a2-b1ad-469e-8141-6583ef1c6c8b" />

**Git** - База, хуй жидам, бесплатно синхронизируемся. 

<img width="359" height="707" alt="image" src="https://github.com/user-attachments/assets/af607061-f964-4b29-8853-2d94df8c75a8" />

**Iconize** - Добавляет иконки к чему угодно, я у себя заюзал для навигации по разделам

<img width="263" height="286" alt="image" src="https://github.com/user-attachments/assets/fd2db416-7def-4e87-8204-024621833aff" />

**Tasks** - Ищет везде синтаксис тасок (- [ ] Something non-important, with no date), и собирает их в одном месте, где можно отслеживать выполнение, глянуть можно подробнее тут https://github.com/obsidian-tasks-group/obsidian-tasks

**Templater** - С помощью этого плагина можно строить темплейты, которые можно юзать например, чтобы создать daily note с нужной структурой, пример:
